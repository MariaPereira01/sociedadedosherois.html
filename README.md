# Sociedade dos Heróis — Site de Críticas e Curiosidades dos Filmes de Super-Heróis

Este projeto é um site desenvolvido para atividade final de Introdução a Web(prática de HTML, CSS e JavaScript).

---

## Funcionalidades

### Layout Completo
- Menu horizontal fixo
- Design com tema escuro
- Uso das fontes **Quicksand** (corpo) e **Montserrat** (títulos)
- Ícones do **Google Material Symbols**

### Estrutura de Páginas
- **Home**
- **Filmes**
- **Categorias**
- **Críticas**
- **Contato**
- **Sociedade dos Heróis** (item especial no menu)

---

## Tecnologias Utilizadas

### **Frontend**
- HTML5  
- CSS3 (Flexbox, fontes importadas, menu horizontal)
- JavaScript (alerta único, manipulação de formulário)

### **Fontes**
- Google Fonts:  
  - **Quicksand** (padrão do site)  
  - **Montserrat** (títulos)

### **Ícones**
- Google Material Symbols
